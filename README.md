# Machine Learning Algorithms: Custom Models vs. Scikit-Learn

**Project Overview**
This repository is dedicated to understanding fundamental machine learning algorithms by building them from scratch. The project includes custom implementations of Principal Component Analysis (PCA), Linear Regression, Logistic Regression, and an Artificial Neural Network (ANN). These custom algorithms were then benchmarked against industry standard libraries like `scikit-learn` to compare their overall accuracy, performance, and training speed.

### Technical Stack
* **Languages:** Python
* **Libraries:** NumPy, Pandas, Matplotlib, Scikit-learn
* **Environment:** Jupyter Notebook
* **Techniques Used:** Forward/Backward Propagation, Gradient Descent, Eigen Decomposition

---

### The Dataset
The dataset used for this project (`sample_data_for_project_final.csv`) consist of **864 samples** and **15,606 variables**. 

There is also a metadata file (`sample_metadata_for_project_final.csv`) that assigns a binary label (either `0` or `1`) to each sample. Before doing any predictive modeling, I mean centered the data so it was ready for analysis. 

---

### Pipeline Workflow

Data Import -> Mean Centering -> Dimensionality Reduction (PCA) -> Model Setup -> Training  -> Testing -> Benchmarking against SKLearn.

---

**Key Takeaways:**
The custom neural network effectively learned the underlying data patterns, shown by a sharp decrease in cost during early training iterations. The SKLearn implementation achieved slightly higher accuracy and a smoother training curve, primarily due to its utilization of advanced built in optimizers (such as Adam) and adaptive learning rates. 

For the linear and logistic regression tasks, the decision boundaries and correlation scores produced by the custom code closely mirrored those generated by SKLearn.

---

### Future Improvements
While the custom models demonstrated strong baseline performance, future iterations of this project could include:
* **Algorithm Optimization:** Implementing the Adam optimizer or adaptive learning rates within the custom ANN scripts to achieve smoother training curves comparable to SKLearn.
* **Robust Validation:** Integrating k-fold cross-validation to further ensure the models generalize well and do not overfit the training data.
* **Bioinformatics Applications:** Applying these foundational algorithms to real-world genomic or clinical datasets, such as drug response screening or gene expression classification, to bridge core machine learning concepts with biotechnology.
